Variable Binding

Atom allows values to be bind to particular name, referred as 'variable binding'.

In atom, variables are immutable, and assignments are not supported.

let name = "Bharat"

However, you can redeclare the variable with same name using concept called shadowing. After using different value, old value will not be accessible.

let name = "Hany"
